C-2语言组成及数据

本章主要介绍c语言的组成及数据

C语言组成及数据

一、C语言的组成

组成：若干文件
文件：若干函数
函数：函数头+函数尾
函数头：函数名+（）
函数体：{}+语句
语句：三类语句：
                1、/* */           （配对出现）
                2、定义语句；
                3、执行语句；

二、数据与数据类型

数据

数据四类：常量、变量、表达式、函数
常量：值不发生变化的量
变量：值随时随地改变的量
表达式：用运算符将数据连接起来的合法式子
函数：调用具有一定功能的函数作为运算量

数据类型：基本类型（内置）、构造类型、指针类型、实类型

1、基本类型：整型int、字符型char、浮点型（单精度float、双精度double）
2、构造类型：数组型、结构体类型（struct）、共用体类型（union）
3、指针类型
4、空类型（void）

标识符

1、保留字（关键字）：共32个（全部为小写），固定含义
   如：int、float、char、if、else、for等
2、预定义标识符：预先定义并且有特定含义的标识符（命名时可以用，但不建议）
   如：scanf、print、include等
3、用户自定义标识符：由用户根据需要定义的标识符
   如：变量名、数组名、函数名等  
   
注:1）用户自定义标识符： **第一个字符必须是字母或下划线**，后面由字母、数字下划线组成。2）大小写有区分，如int与Int

常量四类：整型常量（年龄）、实型常量（入学成绩）、字符型常量（单个字母）、字符串常量（一串）

1、整形常量：三种形式（十进制、八进制、十六进制）
   十进制：是用一串连续的0...9数字，如32767，-32768，9,0，-7
   八进制：是用数字0开头的一串连续合法八进制，如013,05,0777,089（越界）
   十六进制：是用0x或0X（数字零字母x）开头的一串连续合法十六进制，如0x12,0xfff
   注：长整型变量则必须在后面加字母l或L，如0L,34l,0x123L

2、实型常量：两种形式（小数、指数）
   小数形式：由数字和小数点组成。如：123.456，123. ， .456，,0.456
   指数形式：用“E”或“e”后紧跟一个整数表示以10为底幂数。如：1.23E-5
   注：1）字母E或e之前之后必须都有数字，且后必是整数（可以为负整数）。2）字母E或e前后及个数字之间不能有空格。
   举例：1.23e4（对），5.4e0.5（错），.23e5（对），4E （错），3E2（对），3.E2（对）

3、字符型常量：四种形式（常规、转义、八进制、十六进制）
   常规：一个单引号括一个字符。如‘a’，‘A’，‘@’，‘g’。(''，'\'不合法)
   转义：转义字符常量：必须以一个反斜杆"\"开头。（转义不算一个字符）
    \n 换行符
    \t 制表符
    \b 退格符
    \r 回车符
   八进制：\ddd 最多可以有三位八进制数
   十六进制：\xhh 最多可以有两位十六进制（\x为标志）

4、字符串常量：一种形式 ("a" ""皆合法）
   定义：用一对双引号括起来的若干（0或0个以上）字符。
   如："abcdefg" "1234567" "a1b2c3d4"
注：1）每一个字符串尾都有一个字符串结束符 \0   2)C语言中没有字符串变量，一定要区分字符常量和字符串常量。

变量（把常量存储在变量中。先定义后使用）

变量使用规则：先定义后使用
定义格式：数据类型名 变量名列表；
    (1)int i； int j; int k;
    (2)int i,j,k;
    (3)int i = 1, j = 1, k = 9;
注：1）变量必须先定义后使用。int long float double char(与常量类型一致)
    2）变量名属于用户标识符，遵循用户标识符命名规则。（首必是字母或下划线）
    3）在同一“函数体”{}中不能定义同名变量。（一个变量不能多次定义）
    4）同时定义多变量，必须用逗号分隔。
    5）变量可赋初值（初始化），无赋初值，值是随机的。

函数（标准函数 自定义函数）

#include<math.h>
    √7如何表示？    sqrt()  sqrt(7)
    |-5|如何表示？  abs()   abs(-5)

表达式：用运算符将数据连接起来的合法式子

运算符：三个属性：功能、优先级、综合性（运算方向）
3.5 + 1/2 = 3.5 3.5 + (1/2) = 3.5
1.0/2 = 0.5     1/2 = 0
向同类型转，向进度高的转

1、算术运算符及算术表达式： + - *  / %
    （1）求（%）：
        1）两边运算对象必须是整数。
            如：7.0%3   4%3.  不合法
        求值方法：先忽略负号不计求出余数，再将所得余数的负号定成与被除数相同的符号即可。
    （2）单独的常量、变量或函数调用都是C语言的合法表达式。
    （3）凡是表达式都有一个确定的值。

2、赋值运算符及赋值表达式

    赋值运算符：=
    格式：变量名 = 表达式
    赋值运算符的功能：将表达式的值赋给“=”左边的变量
    例如：A=5+8（先计算后赋值）
    注：（1）赋值运算符左边左边只能是一个变量
        如：4=3（非法）、a+(b=3)(合法表达式)
        （2）赋值运算符右边可以为合法表达式
        如：a=b=20、a=(b=10)/(c=2)
        （3）赋值表达式的值就是赋值运算符左边变量的值
        （4）实型数据赋给整型变量时，实型小数舍去；整型数据赋给实型变量时，系统会自动将整型数据转成实型数据。（向同类型、精度高的转）

    复合赋值运算符
    +=、-=、*=、/=、%=、&=、、|=、^=、>>=、<<=(14级，从右向左)
    注：复合赋值运算符的两个运算符中间不能有空格。

3、自增自减运算符（自增自减表达式，从右向左，2级）
注：只能用于变量不能用于常量或表达式。

4、逗号运算符及其逗号表达式（顺序求值）
格式：表达式1，表达式2，表达式3，...
逗号表达式的值就是表达式n（最后一个表达式）的值，求值顺序是从左到右依次求解。

5、强制转化类型（显示转换）2级
格式：
(类型名) 表达式 或(类型名) (表达式) 注：类型名与表达式中间有空格
利用强制类型转换运算符可将一个表达式的值转换成指定的类型，但不改变变量类型。
如：变量x为int型，则表达式(float) x的结果是一个float型，但是x还是int型。
