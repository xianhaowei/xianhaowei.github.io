四、如何实现输出？

“标准输入输出函数”是一些库函数来实现的
printf();
注意：标准函数/库函数使用前必须: #include<头文件名>在包含的程序文件中

标准函数/库函数的学习方法：头文件名+函数 

1. printf()
格式1：printf("格式控制")；
功能：按格式控制指定的格式输出数据
如:
#include
main()
{
    printf("a = 7, b = 10") //双引号是字符串标志，标志不会无缘无故的输出，双引号内容按原模样输出
}

格式2：printf("普通/占位符", 输出列表的值)；
功能：按格式将值输出
注意1：“占位符”以%d开始的，被替换
       “普通字符”原样输出
    2：输出值，先计算后输出
如：
printf("a = %d, b = %d",3,4+7);

占位符字符
（1） d     带符号十进制整数
（2） u     无符号十进制整数
（3） o     无符号八进制整数
（4） X/x   无符号十六进制整数
（5） c     输出一个字符，不输出单引号
（6） s     输出一个字符串，输出时不输出双引号  \0也不输出
（7） f     输出一个实数型，隐含输出6位小数
（8） E/e   输出指数形式的实数型

五、如何实现输入？

scanf()
格式：scanf("普通/占位符"，地址列表)；
如：
int x;
scanf("%d",&x);

考点1：“普通/占位符，”输入时普通字符 必须原样输入

如：
int a,b;
scanf("%d,%d",&a,&b);
输入：3,5
scanf("a=%d,c=%d",&a,&b);
输入：a=3,c=5

考点2：“占位符”，整型输入时4种输入

int a，b；
scanf("%d%d",&a,&b);
输入：
1）3 空格7 回车确认（空格可以多个）
2）3 tab 7 回车确认（tab可以多个）
3）3 回车 7 回车确认 （回车可以多个）
4）3 空格 tab 回车 7 回车确认 （中的空格、tab、回车可以随便顺序，也可以随便多个）
注意：这四种输入针对整型

考点3：“占位符”，字符型输入时1中输入

char ch1, ch2;
scanf("%c%c",&ch1,&ch2);
输入：ab回车
    ch1=a，ch2=b
如果输入：a空格b回车确认，会显示ch1=a，ch2=空格，中间是tab，回车也如此  

即%d不吸收空格，%c吸收空格

六 字符数据的输入输出

putchar()与putchar()

1）putchar()

格式：putchar(字符常量或字符变量)
功能：输出一个字符但不能输出单引号
如：
putchar('a');   a
putchar('\n';)  光标下移
putchar(100);   d
putchar('100'); 错，字符只可放一个

2）getchar()

格式：getchar()
功能：接收一个从键盘输入的字符
注：getchat()没有任何参数，函数返回值就是输入的字符
如：
char a, b;
a = getchar();
b = getchar();
若输入为： 1 <回车>
则变量a的值为字符‘1’，变量b的值为回车符

注：字母、空格、字符均为合法字符。
